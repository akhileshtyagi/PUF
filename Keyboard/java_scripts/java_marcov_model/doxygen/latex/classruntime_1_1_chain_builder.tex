\subsection{Detailed Description}
Wrapper around Chain used to make using the model easier in a real application. 

This class will construct a model with the given parameters. Calling authenticate will cause the newest \mbox{[}user\+\_\+model\+\_\+size\mbox{]} touches to be compared against \mbox{[}auth\+\_\+model\+\_\+size\mbox{]} touches which immediatly precede the touches used in the user\+\_\+model.
